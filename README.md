### Commands

Clear chat
```
[clear]
```

Check bitcoin price
```
[bitcoin]
```

Check covid statistics
```
[covid]
```

Embed Youtube video
```
[yt <url>] ex. [yt https://youtu.be/5sepzZLy-z8]
```

Guess the number
```
[<1, 1000>] ex. [500]
```
